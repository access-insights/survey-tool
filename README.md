# Access Insights Survey Tool

Accessible Survey + Screener MVP for research teams.

## Repository structure
- `apps/web` React + TypeScript + Vite frontend
- `netlify/functions` Netlify Functions backend API
- `db` Supabase schema, RLS, and seed SQL
- `docs` architecture, setup, accessibility, threat model, deployment, testing

## Quick start
```bash
npm install
npm run install:all
npm run dev
```

Open `http://localhost:8888`.

## Scripts
```bash
npm run build
npm run lint
npm run test
npm run test:a11y
npm run test:e2e
```
